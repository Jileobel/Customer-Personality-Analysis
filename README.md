# Customer-Personality-Analysis
A cleaned dataset of a company’s ideal customers. It helps the business to better understand its customers and modify products according to the specific needs, behaviors, and concerns of different types of customers.
This dataset is tab-delimited, meaning Excel has placed all the data into a single column, which has to be delimited
The dataset consists of 2,241 rows. I created a copy of the original sheet and worked with the copy. This prevents loss of its originality. The data was cleaned using an Excel sheet and its respective formulas.
Duplicated rows were deleted, and all missing values were handled using the filter icon, replacing them with the average score of a column. 
Standardised text values in the education and marital_status column.
Converted all data formats into a consistent type, especially for the date into a short date type.Renamed all column headers to a clean and uniform lowercase with no spaces but underscores.
After all this process, the data is cleaned and ready for any further analysis.
